Autonomous Surveillance Vehicle

Abstract:

Designed and developed an AI-powered surveillance vehicle capable of collision avoidance, path following, and real-time environmental monitoring using NVIDIA Jetson Nano. The system integrates deep learning and image processing to autonomously navigate diverse environments for smart cities, warehouses, and security applications.

Detailed Summary:

The Autonomous Surveillance Vehicle is a self-navigating robotic system built to monitor surroundings, detect obstacles, and autonomously follow roads using computer vision and AI-based decision-making. The project was developed by leading a team of four engineers and deployed on NVIDIA Jetson Nano with ROS for real-time communication and control. Collision Avoidance and Path Following: Integrated a dual-camera vision system with real-time object detection using deep learning (TensorFlow and OpenCV) to classify scenarios as free or blocked and maneuver accordingly. Real-Time Surveillance and Monitoring: Enabled image classification and environmental monitoring, allowing the vehicle to detect anomalies and potential security threats. Embedded AI Deployment: Optimized Jetson Nano-based deep learning models to run efficiently for real-time processing. Applications: Can be deployed for urban surveillance, warehouse monitoring, and industrial automation.

Multi-Robot Exploration

Abstract:

Developed a multi-robot exploration framework using a modified Lévy walk strategy and potential field-based collision avoidance to improve area coverage and mapping efficiency. Simulated in MATLAB.

Detailed Summary:

The project focused on improving autonomous exploration by designing a multi-robot system capable of efficient navigation, collaborative mapping, and obstacle avoidance. Intelligent Exploration Algorithm: Implemented a modified Lévy walk strategy to improve coverage while preventing robots from clustering. Collision Avoidance and Path Planning: Combined potential fields with multi-agent coordination. Simulated Environment and Validation: Validated in MATLAB, achieving over 90% map coverage efficiency. Seamless Map Merging: Each robot’s local map was merged into a complete map. Applications: Search and rescue, planetary exploration, and autonomous warehouse logistics.

Real-Time Incident Detection

Abstract:

Developed an AI-powered real-time incident detection system using deep learning and edge AI to identify accidents, hazards, and obstructions with 95% accuracy. Optimized using TensorFlow Lite and deployed on NVIDIA Jetson Nano. A lightweight version was deployed on Arduino Nano 33 BLE using Edge Impulse.

Detailed Summary:

This project improves road safety and traffic management by integrating deep learning-based incident detection with low-power edge AI for real-time decision-making. A CNN was trained on 300,000+ road images using TensorFlow and achieved 95% precision. The model was optimized using TensorFlow Lite, reducing size by 75% while maintaining accuracy. Deployed on Jetson Nano for real-time inference, and on Arduino Nano 33 BLE using Edge Impulse for low-power use. A LoRaWAN-based alert system transmits incident data for early warnings and routing suggestions. The system integrates with intelligent transportation systems and supports scalability.

Road Marking System

Abstract:

Designed a MATLAB-based road marking system using semantic segmentation and image processing to detect and mark lanes on unmarked rural roads. Supports lane-following and improved navigation in unstructured environments.

Detailed Summary:

Semantic segmentation was used to differentiate road surfaces from surrounding regions and identify drivable areas. Canny edge detection and Hough transforms were used to extract lane boundaries and generate virtual lane markings. A lane departure warning mechanism detects vehicle position relative to generated lanes and triggers alerts. Applications include autonomous driving, agricultural automation, and off-road autonomous vehicles.

ReflexDodgeBot: Real-Time Evasion Using Monocular Vision on Embedded Hardware

Abstract:

Built a real-time object detection and evasion system for mobile robots using monocular vision on low-cost embedded hardware. Uses YOLOv8n for detection and monocular depth estimation for reflex-like obstacle avoidance on NVIDIA Jetson Nano.

Detailed Summary:

The system uses YOLOv8n for object detection and a monocular depth approach to estimate distance from RGB input. A Kalman filter tracks object motion and triggers evasive maneuvers based on approach velocity and relative position. The robot uses mecanum wheels for omnidirectional movement. Deployment was optimized for Jetson Nano with TensorRT and FP16 to maintain real-time performance.
